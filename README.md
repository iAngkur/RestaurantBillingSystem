# RestaurantBillingSystem

__Description:__ Simple Billing System for a restaurant. Billing information will be stored in database.

### Database Structure:
* Customers
  * id		
  * Name	
  * Phone	
  * Total	
  * Cashback	
  * Tax	
  * Subtotal	
  * TimeDate
